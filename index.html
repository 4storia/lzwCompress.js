lzwCompress.js

Lossless LZW compression/decompression implemented in JavaScript for strings/JSON/JS objects.

Usage:

// To compress anything from within JS
var compressed = lzwCompress.pack(humongousObj);

// And to decompress it
var original = lzwCompress.unpack(compressed);

Applies LZW compression and JSON key optimization (if applicable) and makes JSON (or any javascript object) smaller for local storage, for sending up to the server, etc.
